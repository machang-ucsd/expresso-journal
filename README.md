A cozy chrome extension for jotting down the wifi details at your favorite coffee shop.  

### Installation
1. Download repo as zip
2. Unpack zip
3. Open chrome and click extensions icon in top right corner
4. Click `Manage Extensions`
5. Enable developer mode
6. Click `Load unpacked" in top left corner
7. Load the folder containing all the files
